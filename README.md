# Pokédex - Desafio 3 Backend 2026.1

Este projeto consiste em uma Pokédex desenvolvida com HTML, CSS e JavaScript, utilizando a [PokeAPI](https://pokeapi.co/) para buscar informações dos Pokémon.

A aplicação exibe o nome e a imagem do Pokémon atual, permitindo navegar entre os Pokémon disponíveis por meio dos botões **Anterior** e **Próximo**.

Esse desafio é parte da trilha de capacitação BackEnd da EngNet.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PokeAPI
